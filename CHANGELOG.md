# Changelog

All notable changes to `kayne-swagger` will be documented in this file.

## [1.0.0] - 2025-12-27

### Added
- Initial release
- BaseDto class with automatic type binding
- SchemaGenerator for OpenAPI schema generation
- SwaggerGenerator for complete spec generation
- Property and Api attributes for clean syntax
- Automatic DTO validation middleware
- Swagger UI integration
- Support for nested DTOs
- Support for response DTOs
- Auto-discovery for Laravel
- Comprehensive documentation

### Features
-  Type-Safe DTOs with PHP 8.1+ type hints
-  80% less boilerplate than l5-swagger
-  Automatic validation from type definitions
-  Modern PHP attributes instead of verbose annotations
-  DRY principle - define once, use everywhere
-  Fast like .NET Swagger generation

## [Unreleased]
